Obtaining extension and transforming file


After downloading the file FBI-R-----, our working directory has the next structure:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/pdf_leaks$ ls
FBI-R-----


To obtain the extension of the file, we will have a look at its magic number with the file command:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/pdf_leaks$ file FBI-R----- 
FBI-R-----: Zip archive data, at least v2.0 to extract


As we can see, we discovered that the downloaded file is a zip archive, and the version of zip we need in order to extract it. In order to extract the files properly, we will change the extension of the file to .zip:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/pdf_leaks$ mv FBI-R----- FBI-R-----.zip
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/pdf_leaks$ ls
FBI-R-----.zip


Let's extract its content with the command unzip <fileName>:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/pdf_leaks$ unzip FBI-R-----.zip
Archive:  FBI-R-----
  inflating: FBI-Report.pdf

After extracting the contents, we obtained a pdf file with the name FBI-Report.pdf.
