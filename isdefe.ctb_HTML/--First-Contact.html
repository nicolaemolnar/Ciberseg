First-Contact


The challenge provides a file with the name ?? whose extension is unknown, so the first step is to obtain the extension of the file with the command file:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/osint$ file \?\? 
??: PNG image data, 546 x 540, 8-bit/color RGBA, non-interlaced

We discovered that the file is actually a PNG image, so we apply the corresponding transformation with the mv command:
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/osint$ mv \?\? image.png
alex@DESKTOP-MQKMDU5:/mnt/c/Ciberseg/osint$ ls
image.png


After these steps, we can open the image with any program of our O.S. and see its content:

The image represents a QR code, this QR, when scanned with a QR Scanner, provides the following string 15otkz1oqgvntjyqf8mj9mnqkineeu4skg.
Notice that this challenge is in the OSINT category, OSINT stands for Open Source INTelligence, so we will take the string obtained from the QR and make a google search with it as a query:

The " character in the google query makes the search retrieve only the results that match exactly with the string, same characters and same order between them. We discover that the string corresponds to a blockchain address, in particular, a bitcoin address:

The QR seen in the image above corresponds to the one provided by the challenge and so does the address, this means we found the page the challenge wanted us to reach. Now, following the principles of OSINT, we need to find clues and data related to the flag.

Flag submission tries:
